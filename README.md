## Project Description
This project is about analyzing an open-source software (Git). It includes study of its origin, license, Linux usage, ecosystem, and comparison with proprietary software. It also includes shell scripting tasks.

## Scripts

### Script 1 – System Identity Report
Displays system details like kernel, user, uptime, and OS.

### Script 2 – Package Inspector
Checks whether Git is installed and shows version.

### Script 3 – Disk Auditor
Shows directory permissions and disk usage.

### Script 4 – Log Analyzer
Counts keyword occurrences in log file.

### Script 5 – Manifesto Generator
Generates a personalized open-source manifesto.

## How to Run Scripts

1. Open terminal
2. Navigate to folder
3. Run:
   chmod +x script1.sh
   ./script1.sh

Repeat for all scripts.

## Requirements
- Linux / Git Bash
- Basic terminal knowledge
