## 📌 Project Description
This project analyzes Git, an open-source version control system. It covers its origin, license, Linux usage, ecosystem, and comparison with proprietary tools.

## 📜 Scripts Description

### Script 1 - System Identity Report
Displays system details like kernel, user, uptime, OS distribution.

### Script 2 - Package Inspector
Checks if Git is installed and shows version and details.

### Script 3 - Disk Auditor
Displays directory size, permissions, and ownership.

### Script 4 - Log Analyzer
Counts errors in system logs and displays last matches.

### Script 5 - Manifesto Generator
Generates a personalized open-source philosophy statement.
